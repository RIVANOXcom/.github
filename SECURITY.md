# Security Policy

Please do not disclose security-sensitive information in public GitHub issues.

## Reporting security concerns

Send security-related information through the official contact channel listed on the RIVANOX website:

https://rivanox.com/legal-notice/

Include:

- affected product
- product version
- platform or environment
- clear description of the concern
- reproduction steps where possible
- potential impact

Do not include sensitive customer data, license keys, passwords, access tokens or private repository contents unless a secure reporting process has been agreed.
