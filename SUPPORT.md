# RIVANOX Support

For public feedback, bug reports, improvement ideas and documentation issues, use the RIVANOX support repository:

https://rivanox.com/support/

## Product-specific support

Each product may have its own support, licensing and update workflow.

| Product | Main support direction |
|---|---|
| WakeAnchor | Product page, license workflow and GitHub issues where available |
| Git/Gitea Sync Monitor | Product page, managed demo and Pro license workflow |
| XML Sitemap Filter | Product page, GitHub issues and Pro support where applicable |
| Multilingual Manager | Product page and GitHub issues where available |

## Before reporting

Please include:

- product name
- product version
- operating system or WordPress version
- exact problem description
- steps to reproduce
- screenshots or logs where useful
- whether the issue concerns download, installation, licensing, update or product behavior

Do not post license keys, private customer data, repository contents, access tokens, passwords or security-sensitive details in public issues.
